My Work

02

BabySitters WeChat MiniApp 📗

This project is provide nanny, childcare related domestic services. There are mainly user registration and user, employee information management module, order module and payment module. A 3-tier architecture is used to play user interface layer, business logic layer and data access layer separately, to reduces the coupling of the code. Use MyBatis to avoid using traditional JDBC and separate SQL from code.

Film Website 📕

Oh what's that? The aim of the FilmWebsite project was to create a web platform for movie lovers. The primary objectives included user authentication, movie-related interactions, content display, search functionality, and an administrator backend system. The project aimed to provide users with a comprehensive movie-related experience while enabling administrators to manage users and movie content efficiently.

Yoke Enterprise Website 📘

This is a simulation of a B2C website developed based on Sass, featuring display functionality and backend login capabilities. The display panel includes product introductions, services, and corporate news. Additionally, features such as fuzzy news search, news categorization, and news details have been added based on demand. In the backend operating system, there are functions for managing user addition, editing, deleting news, as well as statistics for news data created by users.​

Rock Paper Scissors 📙  

This is an embedded program that uses C to program an ATmega32U2 microcontroller called UC FunKit. The program runs an interactive game between two FunKits. The board has a LED display which is the main game interface, and the game is controlled by a nav switch and a button on the board. The two boards communicate via infrared (IR).​
